# Service-Job-Cost-Calculator
A simple C++ program that calculates the total cost for a service job based on user input.   The program prompts for the customerâ€™s name, hours of labor, and cost of parts, then computes labor charges, sales tax, and the final total.
## Features
- Prompts user for:
  - Customer name
  - Hours of labor
  - Cost of parts
- Calculates:
  - Labor cost (fixed rate of $35/hour)
  - Sales tax (5% of parts cost)
  - Total cost
